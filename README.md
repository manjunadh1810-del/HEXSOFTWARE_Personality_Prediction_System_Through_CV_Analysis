# 🧠 Personality Prediction Through CV Analysis

## 📌 Project Overview

This project predicts a candidate's personality traits by analyzing the content of their resume (CV). The system extracts text from the resume and identifies important keywords related to different personality traits.

The project helps understand a candidate's strengths, skills, and personality characteristics using Natural Language Processing techniques.

---

## 🎯 Objective

- Extract text from resumes.
- Analyze keywords and skills.
- Predict personality traits.
- Visualize personality scores.
- Automate resume analysis.

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NLTK
- PyPDF2
- Matplotlib

---

## 📂 Project Workflow

### 1. Resume Upload
The user uploads a resume in PDF format.

### 2. Text Extraction
Text is extracted from the PDF using PyPDF2.

### 3. Text Processing
The extracted text is converted into lowercase and processed.

### 4. Keyword Analysis
Important keywords related to different personality traits are identified.

### 5. Personality Prediction
The system predicts the dominant personality trait based on keyword scores.

### 6. Visualization
Personality scores are displayed using a bar chart.
